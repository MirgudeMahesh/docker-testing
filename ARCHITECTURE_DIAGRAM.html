📦 Component Overview

Client Browser - User's device
Nginx Container - Reverse proxy
React Frontend - Static files
Backend Container - API server
Express API - Business logic
MySQL Database - Data storage

🔑 Key Features

  • Nginx Reverse Proxy: Routes static files and proxies API requests to backend
  • React Multi-stage Build: Optimized production build served as static files
  • Express API: Handles authentication (login/register/logout) and JWT tokens
  • HTTP-Only Cookies: Secure token storage, prevents XSS attacks
  • CORS Protection: Restricted origins, credentials validation
  • MySQL Connection Pool: Efficient database connections
  • Docker Compose: Single command deployment of all services

🚀 Authentication Flow

  • Login: User credentials → Backend validates → JWT token created → Stored in HTTP-only cookie
  • Protected Requests: Browser auto-includes cookie → Backend validates token → Access granted
  • Logout: Cookie cleared → User session terminated → Redirected to login
